解决 hexo 大量空行

Dec 15, 2020

前言

在使用 PageSpeed Insights 时 blog 的分数是 62，一看 html 文件发现大量空行。

开始解决

$ npm install hexo-html-minifier --save

安装 hexo 插件，Github。

在站点 _config.yml ,里添加

html_minifier:
  exclude:

查看 html 文件。

搞定！
