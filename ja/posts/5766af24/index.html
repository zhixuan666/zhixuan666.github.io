Tailwind v2 無理やり対応させる方法

May 24, 2021

Tailwind v2.1.2 からは、IE11を含むIEのどのバージョンも対応しておりません (grid は使えません

対応した v1.9.0 を使いましょう

Tailwind 無理やり対応させる方法

autoprefixer や postcss-color-rgba-fallback を利用する

autoprefixer: 対応対象にしたいブラウザを設定すると、自動的にベンダープレフィックスを付け

/* 使用する前 */
::placeholder {
    color: gray;
}

.image {
    background-image: url(image@1x.png);
}
@media (min-resolution: 2dppx) {
    .image {
        background-image: url(image@2x.png);
    }
}

/* 使用した後 */
::-moz-placeholder {
    color: gray;
}
:-ms-input-placeholder {
    color: gray;
}
::placeholder {
    color: gray;
}
.image {
    background-image: url(image@1x.png);
}
@media (-webkit-min-device-pixel-ratio: 2),
        (min-resolution: 2dppx) {
    .image {
        background-image: url(image@2x.png);
    }
}

postcss-color-rgba-fallback: IE が対応してない　rgba　を16進数カラーコードを追加する

/* 使用する前 */
body {
    background: rgba(153, 221, 153, 0.8);
    border: solid 1px rgba(100,102,103,.3);
}

/* 使用した後 */
body {
    background: #99DD99;
    background: rgba(153, 221, 153, 0.8);
    border: solid 1px #646667;
    border: solid 1px rgba(100,102,103,.3);
}

Install

// yarn: パッケージ管理システム
$ npm install -g yarn
$ mkdir ./new-project

// プロジェクト初期化
$ yarn init -y

// tailwindcss や プラグイン install
$ yarn add tailwindcss autoprefixer postcss-color-rgba-fallback

// tailwindcss config 初期化
$ yarn run tailwindcss init -p

// postcss install
$ yarn add postcss postcss-cli --dev

Config

// tailwind.config.js
module.exports = {
    purge: [],
    darkMode: false, // or 'media' or 'class'
    theme: {
        extend: {},
    },
    variants: {},
    plugins: [],
}

// postcss.config.js
module.exports = {
    plugins: [
        require('tailwindcss'),
        require('autoprefixer'),
        require('postcss-color-rgba-fallback')
    ]
}

Build

$ postcss src/style.css -o src/style.min.css
