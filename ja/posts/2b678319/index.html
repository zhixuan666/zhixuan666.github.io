linux Minecraftサーバーを構築

Sep 21, 2020
文章阅读量 1000000

初めに

{% note info %}

この記事では、全体で root ユーザーを使用して Linux でビルドする方法のみを説明し、使用されるディストリビューションはDebian9.5です。 Ubuntu やその他の Debian ベースの Linux にも適しています。RHELは一部のコマンドのみが異なり、他のコマンドは同じです。

{% endnote %}

Minecraftサーバーを構築

Java と Screen インストール

Ubuntuおよびその他のDebianベースのコマンド

$ sudo apt-get update
$ sudo apt-get install java screen -j

RHELのコマンド

$ sudo yum update
$ sudo yum install java screen -y

サーバーにファイルを保存するためのディレクトリを作成します

cd ~
mkdir mcserver
cd /mcserver

forge サーバーを構築するには、こちら

普通 サーバーを構築する

MCversions にアクセスし

必要なバージョンの Server jar を右クリックし、リンクをコピーして、ターミナルへ

$ wget コピーしたリンクを貼り付けます

ダウンロード後、サーバーを起動します。

$ java -Xms512M -Xmx1024M -jar server.jar -nogui

{% note info %}

初めては　EULA ファイルを変更する必要があります

$ vim eula.txt

eula = falseをeula = trueに変更し、コマンドを再入力します。

{% endnote %}

構成ファイルについて

forge サーバーを構築する

MinecraftForge にアクセスし

左がバージョン、真ん中が最新バージョン、右が安定版です。ここから安定版をダウンロードします。

右クリックし、リンクをコピーして、ターミナルへ

$ wget コピーしたリンクを貼り付けます
$ java -jar forge-*****-install.jar -nogui --installServer

インストールを待っていると、 ls　はさらにいくつかのファイルを確認し、次のように入力します。

$ java -Xms512M -Xmx1024M -jar forge-*****-universal.jar -nogui

{% note info %}

初めては　EULA ファイルを変更する必要があります

$ vim eula.txt

eula = falseをeula = trueに変更し、コマンドを再入力します。

{% endnote %}

{% note info %}

modを追加する方法

ダウンロードしたmodをmodsフォルダーに入れて、サーバーを再起動。

{% endnote %}

構成ファイル server.properties

enable-command-block=true   #コマンドブロックを有効にする
query.port=25565            #サーバポート
pvp=false                   #PVPをオフにします
